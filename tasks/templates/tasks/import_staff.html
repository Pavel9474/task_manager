Импорт штатного расписания

Инструкция по импорту:
  1. Загрузите файл штатного расписания в формате Excel (.xlsx или .xls)
  2. Файл должен содержать колонки:
    • Структурное подразделение - иерархия подразделений
    • ПКГ - профессиональная квалификационная группа
    • Должность - название должности
    • ФИО - фамилия, имя, отчество сотрудника
    • Ставка - размер ставки (1, 0.5, 0.25)
    • Тип занятости - основное, совмещение и т.д.
  3. Будет создана иерархическая структура подразделений
  4. Для каждого сотрудника будет создана запись или обновлена существующая
